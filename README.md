# Ensemble Learning-Based XGBClassifier for Loan Approval Prediction

## 📌 Project Overview
This project delivers an end-to-end machine learning pipeline built to automate financial risk assessment and minimize credit defaults for lending institutions. By leveraging ensemble learning, the system evaluates applicant profiles in real-time to determine loan eligibility with exceptionally high precision.

## 📊 Model Performance
* **Training Accuracy:** 99.57%
* **Test Accuracy:** 96.78%

## 🛠️ Key Features & Workflow
1. **Data Preprocessing:** Handled missing values, executed Label Encoding for categorical variables, and applied Feature Scaling to optimize algorithm convergence.
2. **Feature Engineering:** Analyzed critical financial metrics including applicant income, CIBIL scores, asset valuations, and loan amounts to identify primary risk drivers.
3. **Advanced Modeling:** Trained an **XGBoost Classifier (XGBClassifier)** to exploit gradient-boosted decision trees for superior classification boundaries.
4. **Predictive Runtime System:** Integrated a real-time prediction pipeline utilizing **NumPy** to reshape and process incoming applicant vectors for instant risk classification.

## 💻 Tech Stack
* **Language:** Python
* **Libraries:** XGBoost, Scikit-Learn, NumPy, Pandas
* **Environment:** Google Colab / Jupyter Notebook
